Google Music Manager implemented in C#

This repository contains C# source to enable uploading and downloading of tracks to Google Music. The GoogleMusicManagerAPI is the main assembly, and GoogleMusicManagerCLI and GoogleMusicDownloaderCLI are example command line interfaces showing to upload and download.
